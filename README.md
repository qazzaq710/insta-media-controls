# Insta Media Controls

**Insta Media Controls** is a Chrome extension that makes browsing Instagram more convenient by adding better controls for videos, photos, stories, and carousels.

## Features

- Adds a convenient bottom media control panel
- Lets you seek through Instagram videos
- Shows video progress and duration
- Adds volume control
- Opens photos, videos, and stories in a clean fullscreen mode
- Shows only the content itself, without the Instagram interface, chat, or side panels
- Supports carousels: browse photos and videos directly in fullscreen mode

## Installation

1. Download or clone this repository.
2. Open Chrome and go to: `chrome://extensions/`
3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select the folder with the extension files.

## Usage

Open Instagram and start watching a video, story, photo, or carousel.

The extension will add extra media controls directly on the page.

## Privacy

This extension works locally in your browser.

It does not collect, store, or send any personal data.
